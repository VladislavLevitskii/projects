# Zápočtový program

Hra **Člověče, nezlob se!**

## Specifikace

Tento program implementuje hru **Člověče, nezlob se!** pro 2 až 4 hráče. Hráči se pohybují po herní desce, hází kostkou a snaží se dostat své figurky do cíle. Program je napsán v Pythonu a využívá knihovny `tkinter` a `pillow` pro grafické uživatelské rozhraní.

## Instalace a spuštění

Pokud máte nainstalovaný Python 3.x, stačí spustit hlavní soubor `main.py`. K tomu použijte následující příkaz:

```
python main.py
```

Pokud nemáte knihovny `tkinter` nebo `pillow`, můžete nainstalovat pomocí:

```
pip install tk
pip install pillow
```
